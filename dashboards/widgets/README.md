# Widgets

Widgets are the tiles on a [dashboard](../dashboards.md) grid. Each one displays a different type of data or content from your cameras. The [Dashboards overview](../dashboards.md#how-dashboards-connect-to-the-rest-of-the-system) explains how widgets connect to the rest of the system.

To add, move, resize, or update widgets, follow the steps in [Create and manage dashboards](../create-and-manage-dashboards.md).

## Widget types

| Widget                            | What it does                                                                   |
| --------------------------------- | ------------------------------------------------------------------------------ |
| [Chart or table](chart-or-table/) | Visualizes object detections, alert counts, or event tags as a chart or table. |
| [Heatmap](heatmap.md)             | Shows where activity is concentrated in a camera's field of view.              |
| [Occupancy](occupancy.md)         | Tracks how many people are in a space at any point in time.                    |
| [Image](image.md)                 | Displays a static image, such as a floor plan or reference photo.              |
| [Text](text.md)                   | Adds a formatted text block for labels, descriptions, or context.              |

## Working with widgets

All widget management tasks are covered in [Create and manage dashboards](../create-and-manage-dashboards.md): adding, arranging, resizing, changing settings, renaming, saving, and deleting.

Widgets can't be duplicated. If you need a similar tile, add a new widget and configure it from scratch.
