# Text

The Text widget adds a formatted text block to your dashboard. Use it to label a group of widgets, describe what a section of the dashboard is showing, or add any written context that helps your team make sense of the data at a glance.

## Add a Text widget

1. Open a dashboard and select **Add widget**.
2. Select **Text**. The text editor dialog opens.

3. Enter your text in the editor area.
4. Use the toolbar to format it as needed.
5. Select **Add** to place the text block on the dashboard canvas.

## Formatting options

The toolbar gives you the following formatting controls:

- **Paragraph styles**: Set text as a heading, subheading, or body paragraph.
- **Bold**, **italic**, and **underline**.
- **Font size** and **font color**.
- **Highlight color**: Apply a background color to selected text.
- **Alignment**: Left, center, or right.
- **Bullet list** and **numbered list**.
- **Link**: Add a hyperlink to selected text.

## Resize the widget

With the dashboard in **edit mode** (select the **edit icon** / pencil in the top right), drag the edges or corners of the widget to resize it. The text reflows automatically as you resize.

## Edit or delete the widget

To edit the text:

1. Select the **edit icon** (pencil) in the top right — tooltip **Edit dashboard**.
2. Select the **edit icon** on the widget.
3. Update the text and select **Save**.

To delete the widget, select the **delete icon** while in edit mode.
