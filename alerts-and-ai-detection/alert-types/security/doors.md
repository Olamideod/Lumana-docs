# Door state change

The door state change alert notifies you when a door opens or closes in the camera view. It's a lightweight way to monitor access points without needing physical door sensors.

## How it works

Lumana monitors the camera view for changes in door position. When the state changes, the alert fires and captures footage of the transition. You can configure it to trigger on either direction, such as a door opening when it shouldn't, or a door that's been left open.

## When to use it

This alert is useful whenever you need to track door activity from a camera feed alone, without dedicated hardware at the door.

- Monitoring after-hours access to restricted rooms or storage areas
- Detecting doors left open in climate-controlled environments like server rooms or cold storage
- Auditing entry and exit at access-controlled points without dedicated door sensors
