# Vehicle alerts

Lumana can detect and track vehicles across your cameras and trigger alerts based on their presence, identity, or behavior. Vehicle-specific alerts are available across the Security, Identification, and Tracking categories.

## Alert types

The alerts below are organized by category. Choose the one that matches what you're trying to detect.

### Security

Security alerts trigger based on a vehicle's behavior or presence.

- **Motion**: Triggers when a vehicle moves within a defined zone or across the camera view.
- **Proximity**: Triggers when a vehicle is within a defined distance of a person or another vehicle for a set duration.
- **Speed control**: Triggers when a vehicle moves faster than a configured speed threshold.
- **Zone protection**: Triggers when a vehicle enters a defined zone and remains for a set duration.

### Identification

Identification alerts trigger based on a vehicle's license plate.

- **License plate recognition**: Triggers when a vehicle with a specific plate appears or does not appear on a defined list. Use it to flag unauthorized vehicles or monitor access by known ones.

### Tracking

Tracking alerts monitor how a vehicle moves relative to defined zones, lines, and paths.

- **Appearing**: Triggers when a vehicle enters a defined zone.
- **Disappearing**: Triggers when a vehicle leaves a defined zone.
- **Line crossing**: Triggers when a vehicle crosses a defined line.
- **Traffic control**: Triggers when a vehicle completes a designated path within a configured time frame.

Choose the category that best fits what you're trying to detect, then follow the steps below to create the alert.

## Configure a vehicle alert

The steps are the same regardless of which alert type you choose. When prompted to select an object type, select **Vehicle**.

1. Select **Alerts** in the left navigation bar, then select **Configurations**.
2. Select **Add alert**.
3. Select the relevant category: **Security**, **Identification**, or **Tracking**.
4. Select an alert type and follow the configuration steps.
5. When prompted to select an object type, choose **Vehicle**.
